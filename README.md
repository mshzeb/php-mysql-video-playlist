# PHP Video Playlist Web App with MySQL

## Features

Filter Videos based on Time like:
1. 1 Hour Ago
2. 4 Hour Ago
3. 1 Day Ago
4. 1 Week Ago
5. 14 Days Ago (2 Week)
6. 1 Month Ago

And also based on user like which user uploaded the videos on which time slots.

### Technoloy Stack:
1. PHP
2. MySQL
3. AJAX
